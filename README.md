Add this line to your application's Gemfile:

    gem 'tic-tac-toe'

And then execute:

    $ bundle

    $ gem install tic-tac-toe

To run Human to Human game:

    $ ruby bin/run.rb H

To run Human to Computer game:

    $ ruby bin/run.rb
